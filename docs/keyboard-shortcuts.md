# PufferChat Keyboard Shortcuts

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Search rooms & messages |
| `Alt+↑` | Previous room in list |
| `Alt+↓` | Next room in list |
| `Ctrl+T` | Focus message composer |
| `Escape` | Close current panel / dialog |
| `Tab` | Navigate between UI elements |

## Messaging

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line |
| `↑` (empty composer) | Edit last message |
| `Ctrl+B` | Bold text |
| `Ctrl+I` | Italic text |
| `Ctrl+U` | Underline text |
| `Ctrl+Shift+C` | Inline code |
| `Ctrl+V` | Paste (including images) |

## Rooms

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | Create new room |
| `Ctrl+Shift+N` | Join room by address |
| `Ctrl+,` | Open settings |

## Calls

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+M` | Toggle microphone mute |
| `Ctrl+Shift+E` | Toggle camera |
| `Ctrl+Shift+S` | Toggle screen share |
| `Space` (hold) | Push-to-talk (when enabled) |

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl+/` | Show this shortcuts list |
| `Ctrl+Shift+D` | Toggle dark mode |
| `F11` | Toggle fullscreen |
